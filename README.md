# softmatterlab
Website for Soft Matter Lab at Ashoka University.

**Add/Edit content:** To add/edit website content, go to 'Data' forlder -> choose jason file for content you want to edit -> Add/edit content following existing format. (Alternatively, you can easily manage or edit website content by logging into Softmatter lab's 'Cloudcanon' account.)

**Image Files:** Upload all image files in 'Images' folder. Make sure the image name matches name mentioned in respective conent block.


**DO NOT CHANGE CONTENT FORMAT, OR EDIT ANY HTML OR CSS FILES**
